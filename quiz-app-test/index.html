About This Project

This Interactive Quiz Application was developed as a university project to demonstrate modern web development skills. It features a responsive UI built with pure HTML, CSS, and JavaScript, paired with smooth animations for a premium user experience.

Under the hood, the application is securely connected to a live PostgreSQL database hosted on Supabase and is deployed using Vercel's Serverless Functions to handle real-time leaderboard tracking and scoring.
